# 🛰️ Satellite Map Downloader

یک برنامه گرافیکی برای دانلود تصاویر ماهواره‌ای با کیفیت بالا از منطقه‌ی مورد نظر شما.

## ✨ ویژگی‌ها

- دانلود تصاویر ماهواره‌ای از **ESRI World Imagery**
- انتخاب **مختصات** (طول و عرض جغرافیایی)
- تنظیم **شعاع** (تا ۵۰ کیلومتر)
- انتخاب **سطح زوم** (۱۰ تا ۱۹)
- خروجی در فرمت‌های **PNG، JPG، WebP**
- کنترل **کیفیت** تصویر
- تنظیم **ابعاد** خروجی
- دانلود **همزمان** برای سرعت بالا
- **تلاش مجدد** خودکار برای تایل‌های شکست‌خورده
- رابط کاربری **زیبا و تاریک** با Tkinter
