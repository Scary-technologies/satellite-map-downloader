# 🛰️ Satellite Map Downloader

یک برنامه گرافیکی برای دانلود تصاویر ماهواره‌ای با کیفیت بالا از منطقه‌ی مورد نظر شما.

## ✨ ویژگی‌ها

- **انتخاب مختصات با کلیک روی نقشه** (تعاملی)
- دانلود تصاویر ماهواره‌ای از **ESRI World Imagery**
- تنظیم **شعاع** (تا ۵۰ کیلومتر)
- انتخاب **سطح زوم** (۱۰ تا ۱۹)
- خروجی در فرمت‌های **PNG، JPG، WebP**
- کنترل **کیفیت** تصویر
- تنظیم **ابعاد** خروجی
- دانلود **همزمان** برای سرعت بالا
- **تلاش مجدد** خودکار برای تایل‌های شکست‌خورده
- رابط کاربری **زیبا و تاریک** با نقشه تعاملی

## ⚠️ نکته مهم برای کاربران ایران

به دلیل محدودیت‌های شبکه، کاربران داخل ایران برای دانلود تصاویر ماهواره‌ای **حتماً باید از VPN استفاده کنند** و همچنین برای نمایش نقشه در برنامه، نیاز به اتصال اینترنت بدون محدودیت دارید.
